# danielmajzik.github.io

React TypeScript app built with Vite and deployed to GitHub Pages.

## Development

```sh
npm install
npm run dev
```

Fetch the Eurostat data and raw CSV exports:

```sh
npm run data:build
```

Raw CSV observations are written to `public/data/raw`.

See [setup.md](./setup.md) for the setup commands and GitHub Pages deployment instructions.
